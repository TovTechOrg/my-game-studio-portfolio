# Prompt used

Your role is a professional game developer educator. You need to make several HTML based games and put each in its distinct folder.

The first game is Rock Paper Scissors. This game will be implemented using Vanilla JavaScript for logic and CSS Media Queries for automatic Dark Theme support. The game is designed as an educational piece to demonstrate basic DOM manipulation and event handling.Key Implementation Details:
Technology: HTML5, CSS3, and Vanilla JavaScript.
Game Logic: The game is played against a computer and tracks scores up to a maximum of 3 wins.
Automatic Dark Theme: CSS utilizes @media (prefers-color-scheme: dark) to automatically switch the entire UI's color scheme based on the user's operating system preferences.
User Experience:
Buttons are used for player choices.
A dedicated message area displays the outcome of each round and the final result.
A "Play Again" (Reset) button appears only when the game concludes.
